Hero Modules

Hero modules use the following classes:

Content Container:
content container classes: .content-container .g-py-section .g-px-mobile

Content items:

Main content items should be header H1
H1 .title-level-1
H1 .subtitle-level-1
‍
should use class .g-mb-none to disable any trailing margin on H1s.
